JavaScript Class Syntax

Use the keyword class to create a class. Always add a method named constructor():

	Syntax : 
		class ClassName {
  			constructor() { ... }
		}

		Example
			class Car {
  				constructor(name, year) {
    			this.name = name;
    			this.year = year;
  				}
			}
