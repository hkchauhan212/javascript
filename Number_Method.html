JavaScript Number Methods

The toString() Method

The toString() method returns a number as a string.

All number methods can be used on any type of numbers (literals, variables, or expressions):

The toString() method converts a number to a string.


The toExponential() Method

The toExponential() method returns a string, with the number rounded and written using exponential notation.

An optional parameter defines the number of digits behind the decimal point.

document.getElementById("demo02").innerHTML = y.toExponential() + "" + x.toExponential(2) + "" + y.toExponential(4) + "" + y.toExponential(6);


The toFixed() Method

The toFixed() method rounds a number to a given number of digits.

For working with money, toFixed(2) is perfect.

document.getElementById("demo03").innerHTML = z.toFixed(0) + "" + z.toFixed(2) + "" + z.toFixed(4) + "" + z.toFixed(6);


The toPrecision() Method

The toPrecision() method returns a string, with a number written with a specified length:

document.getElementById("demo04").innerHTML = a.toPrecision() + "" + a.toPrecision(2) + "" + a.toPrecision(4) + "" + a.toPrecision(6);


The valueOf() Method

The valueOf() method returns a number as a number:

document.getElementById("demo05").innerHTML = b.valueOf() + "" + (123).valueOf() + "" + (100 + 23).valueOf();

The parseInt() Method

parseInt()

parseInt() parses a string and returns a whole number. Spaces are allowed. Only the first number is returned:

The global JavaScript function parseInt() converts strings to numbers:

document.getElementById("demo10").innerHTML = parseInt("-10") + "" + parseInt("-10.33") + "" + parseInt("10") + "" + parseInt("10.33") + "" + parseInt("10 6") + "" + parseInt("10 years") + "" + parseInt("years 10");


The parseFloat() Method

parseFloat() parses a string and returns a number. Spaces are allowed. Only the first number is returned:

The parseFloat() method converts strings to numbers:

document.getElementById("demo11").innerHTML = parseFloat("10") + "" + parseFloat("10.33") + "" + parseFloat("10 6") + "" + parseFloat("10 years") + "" + parseFloat("years 10");


Number Object Methods

The isInteger() method returns true if the argument is an integer.

Otherwise it returns false.


The isSafeInteger() Method

The isSafeInteger() method returns true if the argument is a safe integer.

Otherwise it returns false.
